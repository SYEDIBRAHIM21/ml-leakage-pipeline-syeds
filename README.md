# ML Leakage Pipeline Project

## Objective
Build a regression model and evaluate performance.

## Features
- area_sqft
- num_bedrooms
- age_years

## Model
Multiple Linear Regression

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Result
Model performs well with high R² and low error values.

## Conclusion
Residuals are centered around zero, indicating no bias and no data leakage.
